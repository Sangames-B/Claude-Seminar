CLAUDE SEMINAR - SLIDE DECK IMAGE ASSETS INSTRUCTIONS
========================================================================

To display your actual profile picture and LinkedIn QR code on the final slide:

1. PROFILE PICTURE:
   - Save your photo as "profile.jpg" (in JPG format).
   - Place the file inside this "assets" directory.
   - Size recommendation: Square ratio, e.g., 300x300 pixels or larger.

2. LINKEDIN QR CODE:
   - Download your official LinkedIn QR code image.
   - Save it as "linkedin_qr.png" (in PNG format).
   - Place the file inside this "assets" directory.

FALLBACK MODE:
If these files are missing, the presentation will automatically use:
- An elegant Unsplash avatar placeholder for your picture.
- A live-generated QR code linking to LinkedIn's main page.
